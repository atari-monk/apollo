# Project setup

[Up](index.md)
[Main](../../../../index.md)

- Create project

```bash
npx create-nx-workspace@latest helios
```

- Install js plugin

```bash
npm install --save-dev @nrwl/js
```

- Create library

```bash
nx g @nx/js:library engine --directory packages/engine
```
